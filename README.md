# Bots

### Изготовление
1) плоские детали вырезаются из материала толщины 3мм

2) шестеренки, уголки и цилиндры печатаются на 3d-принтере

3) деталь, испытывающую наибольшую нагрузку (Link arm drive) лучше напечатать из пластика, сделав расширенние точки креления к валу

4) валы с фаской в форме полумесяца лучше изготовить из металлического стержня стержня диаметром 4мм или болта М4
